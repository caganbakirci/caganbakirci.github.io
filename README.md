# caganbakirci.github.io

Minimal personal site for Cagan Bakirci (GitHub Pages).

## How to deploy
1. Create a **public** repo named `caganbakirci.github.io` on GitHub.
2. Add these files in the repo root:
   - `index.html`
   - `headshot.jpg`  ← your cropped headshot
   - `.nojekyll` (empty file, optional but harmless)
   - *(optional)* `cv.pdf` to enable the CV button.
3. In **Settings → Pages**, set Source to **Deploy from a branch → main**.
4. Your site will be at `https://caganbakirci.github.io`.

## Custom domain (optional)
- Buy `caganbakirci.com` (or similar) and add it in **Settings → Pages → Custom domain**.
- In your DNS, either point **A** records to GitHub Pages IPs or set a **CNAME** to `caganbakirci.github.io`.
